## Projects

### 1. Digital Clock
A real-time digital clock displaying current time and date.

**Features:**
- Live time display in HH:MM:SS format
- Current date display
- Beautiful gradient background with glassmorphism effect
- Responsive design for mobile and desktop
- Smooth animations

**How to use:**
1. Open `digital-clock/index.html` in your browser
2. The clock updates automatically every second

### 2. Quote Generator
An inspirational quote generator with multiple features.

**Features:**
- Random quote generation from a collection
- Copy quote to clipboard
- Share quotes directly to Twitter
- Beautiful card-based UI with animations
- Responsive design
- Smooth fade transitions between quotes

**How to use:**
1. Open `quote-generator/index.html` in your browser
2. Click "New Quote" to get a random quote
3. Use "Copy" to copy the quote to your clipboard
4. Use "Tweet" to share the quote on Twitter

### 3. Password Generator
A secure password generator with customizable options.

**Features:**
- Adjustable password length (6–64 characters)
- Toggle uppercase, lowercase, numbers, and symbols
- Live strength indicator (Weak / Fair / Good / Strong)
- One-click copy to clipboard
- Generates a password on load

**How to use:**
1. Open `Password Generator/index.html` in your browser
2. Set your desired length and character options
3. Click "Generate Password"
4. Click the copy button to copy to clipboard

### 4. Color Picker Tool
An interactive color picker with multiple format outputs.

**Features:**
- Native OS color picker
- RGB sliders for manual color mixing
- Live HEX, RGB, and HSL code display
- One-click copy for each format
- Recent colors palette (last 10 colors)

**How to use:**
1. Open `Color Picker Tool/index.html` in your browser
2. Click "Pick a Color" or drag the RGB sliders
3. Copy the HEX, RGB, or HSL value with the Copy button
