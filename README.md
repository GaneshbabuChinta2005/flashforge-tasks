## Projects

### 1. Digital Clock
A real-time digital clock displaying current time and date.

**Features:**
- Live time display in HH:MM:SS format
- Current date display
- Beautiful gradient background with glassmorphism effect
- Responsive design for mobile and desktop
- Smooth animations

**Live Demos:**
- Vercel: [https://digital-clock-phi-orpin.vercel.app](https://digital-clock-phi-orpin.vercel.app)
- GitHub Pages: [https://ganeshbabuchinta2005.github.io/flashforge-tasks/digital-clock/](https://ganeshbabuchinta2005.github.io/flashforge-tasks/digital-clock/)

**How to use:**
1. Open `digital-clock/index.html` in your browser
2. The clock updates automatically every second
3. Or visit the live demos above

### 2. Quote Generator
An inspirational quote generator with multiple features.

**Features:**
- Random quote generation from a collection
- Copy quote to clipboard
- Share quotes directly to Twitter
- Beautiful card-based UI with animations
- Responsive design
- Smooth fade transitions between quotes

**Live Demos:**
- Vercel: [https://build-a-quote-generator.vercel.app](https://build-a-quote-generator.vercel.app)
- GitHub Pages: [https://ganeshbabuchinta2005.github.io/flashforge-tasks/quote-generator/](https://ganeshbabuchinta2005.github.io/flashforge-tasks/quote-generator/)

**How to use:**
1. Open `quote-generator/index.html` in your browser
2. Click "New Quote" to get a random quote
3. Use "Copy" to copy the quote to your clipboard
4. Use "Tweet" to share the quote on Twitter
5. Or visit the live demos above

3. Open `index.html` in your web browser
